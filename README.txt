Los archivos QA_barrido e inspector necesitan conexión a los ordenadores de D-Wave apra poder ejecutarse. Además, los QA_barrido consumen alrededor de un 60% del tiempo de computación.
Los archivos inspector_ abren el inspector de problemas para un problema determinado.
matricinator escribe las matrices de interacción usadas en inspector y QA_barrido que luego se usarán para hacer los QUBO correspondientes.
montecarlo usa los valores de beta definidos en parámetros.
repersentación sistema flechas hace una representación del sistema usando flechas para representar espines.
representación temperatura y Representaciones se han usado para representar los resultados de las simulaciones, pero a la hora de hacer los ajustes he preferido usar el programa en Mathematica.
